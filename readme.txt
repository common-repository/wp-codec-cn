== Description ==

Encode / decode the posts and comments with one of the following algorithms: base64, phpjsrsa, rune word etc.

== Installation ==

1.Upload to your plugins folder, usually `wp-content/plugins/` and unzip the file, it will create a `wp-content/plugins/wp-codec-cn/` directory.

2.Activate the plugin on the plugin screen.

3.Done.

== Uninstallation ==

1.Go into admin->plugins ,disable wp-codec-cn.

2.Done.
